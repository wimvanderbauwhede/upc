Welcome to upc.
The aim of this project is to create a compiler that will autmatically compile statically typeable portions of Perl 5 scripts. The main purpose of upc is to speed up execution of computationally intensive code.

The upc will automatically compile statically typeable portions of Perl 5 scripts.

Simply mark a function with use static; and the magic will happen.

This project is at the "idea" stage. Have a look at my Glasgow.pm talk and get in touch if you're interested.
